Filtered Backbone Collections

Documentation
-------------

Contributing
------------

Please open a Pull Request with your changes (including tests).

### Running tests

You will need PhantomJS and NodeJS (including NPM) to run the tests.

```
$ npm install # just once
$ npm test # every time
```
